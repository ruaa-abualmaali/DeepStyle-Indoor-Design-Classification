# DeepStyle-Indoor-Design-Classification
This project focuses on indoor scene classification using multiple deep learning architectures, including CNNs, MobileNet, a custom EfficientNet-B0 variant, and Vision Transformers (ViT). The goal is to evaluate and compare the performance of different modern neural network models on the task of recognizing indoor environments and design-related visual patterns.

Each model is implemented and trained separately in dedicated notebooks, allowing systematic experimentation and performance analysis. The project includes preprocessing, training, evaluation, and prediction generation steps, along with exported results in a CSV file.

A key objective of this work is to analyze how different architectures perform on the same dataset in terms of accuracy, generalization, and computational efficiency. This makes the project valuable for understanding trade-offs between traditional convolutional networks and transformer-based approaches in computer vision tasks.

The final report summarizes experimental findings and provides insights into model behavior and performance differences.
